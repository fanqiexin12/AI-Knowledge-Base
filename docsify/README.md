# docsify Visualization

This folder contains the docsify setup for visualizing your wiki.

## Quick Start

**Local Preview** (recommended for best experience):
```bash
cd docsify
npx serve .
```
Then open `http://localhost:3000`

**Or open directly**:
```bash
open index.html
```
(Note: Some features may not work without a local server due to CORS)

## GitHub Pages Deployment

To deploy on GitHub Pages:

1. Go to repository **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/docs**
4. Save

Then your wiki will be live at: `https://fanqiexin12.github.io/AI-Knowledge-Base/`

## Note

Rename this folder from `docsify` to `docs` before GitHub Pages deployment, or update GitHub Pages settings to point to `docsify`.
