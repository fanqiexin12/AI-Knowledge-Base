# Novel Pb-Free Superconducting Joint Between NbTi and Nb3Sn Wires Using Nb3Hf Intermedia

## Summary
本文提出了一种 **无铅超导接头** 技术，使用 **Nb₃Hf 作为中间层** 连接 NbTi 和 Nb₃Sn 超导线。接头电阻 < 0.5 nΩ，工作温度可达 12T 以上。

## Key Findings

### Joint Design
- **Nb₃Hf 中间层**：高临界磁场 (Hc2)，与 NbTi/Nb₃Sn 扩散焊接
- **无铅设计**：避免传统 Sn-Pb 焊料的环境问题
- **工艺**：热等静压 (HIP) + 扩散退火

### Performance
| Parameter | Value |
|-----------|-------|
| **Joint resistance** | < 0.5 nΩ |
| **Critical current (Ic)** | ~1000 A |
| **Operating field** | > 12 T |
| **Temperature** | 4.2 K |

### Nb3Hf Properties
- **Tc** ~ 8-9 K
- **Hc2** ~ 20-25 T（优于 NbTi）
- **与 NbTi/Nb₃Sn 热膨胀系数匹配良好**

### Advantages over Sn-Pb Joint
- **环境友好**：无铅
- **更高工作温度/磁场**
- **更好的热循环稳定性**
