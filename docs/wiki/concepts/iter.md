# ITER (International Thermonuclear Experimental Reactor)

## Summary
**ITER（国际热核聚变实验堆）** 是正在法国建造的**全球最大国际合作聚变项目**，目标是首次实现 **Q=10**（聚变输出能量是输入能量的10倍）。ITER 采用 **tokamak（托卡马克）** 装置，利用超导磁体产生强磁场约束高温等离子体。

## Key Parameters

| 参数 | 值 |
|------|-----|
| **Fusion power** | 500 MW |
| **Q factor** | 10 |
| **Plasma current** | 15 MA |
| **Magnetic field** | 5.3 T (toroidal) |
| **Major radius** | 6.2 m |
| **Minor radius** | 2.0 m |
| **Construction** | Cadarache, France |

## Superconducting Magnet System

### TF (Toroidal Field) Coils
- **18 TF coils** 围绕等离子体
- **Nb₃Sn** 超导体（高场区 13 T）
- **Cryogenic temperature**：1.8 K (superfluid helium)

### CS (Central Solenoid)
- **6 模块**的 pulsed 超导磁体
- **Nb₃Sn** 制造
- **负责等离子体电流感应**

### PF (Poloidal Field) Coils
- **6 PF coils** 控制等离子体形状和位置
- **NbTi** 超导体（低场区）

### Main Busbar
- **NbTi 超导体**
- **连接 TF 磁体和电源系统**
- **13 kA 级导体**

## NbTi Usage at ITER
- **6 T 以下场区**使用 NbTi
- **Main busbar、PF coils、 Correction coils**
- **Cable-in-Conduit Conductor (CICC)** 设计
